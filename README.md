# Tandem Health Portal


## Getting started

``` shell
git clone https://github.com/bguids91/patient-portal.git
cd patient-portal
bundle
cd client
npm install
cd ..
cd emr
npm install
cd ..
rake db:migrate
rake db:seed
rake start
```
